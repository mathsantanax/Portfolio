Relógio Mundial

Este projeto é uma aplicação web que oferece três funcionalidades principais: Relógio Mundial: Exibe a hora atual de diferentes fusos horários ao redor do mundo. Inclui tanto um relógio digital quanto um analógico, e permite a navegação entre diferentes fusos horários. Cronômetro: Um cronômetro simples que permite iniciar, pausar e resetar o tempo, com precisão de milissegundos. Temporizador: Um temporizador customizável onde o usuário pode definir horas, minutos e segundos, com opções para iniciar, pausar e resetar.
